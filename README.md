# Tic Tac Toe 🎮

A simple Tic Tac Toe game built using HTML, CSS, and JavaScript.

## Features
- Two-player gameplay
- Winner detection
- Draw detection
- Reset Game

## Technologies
- HTML
- CSS
- JavaScript

## How to Run
1. Download the project.
2. Open `index.html` in your browser.
3. Start playing.
