# Tic Tac Toe 🎮

A simple Tic Tac Toe game built using HTML, CSS, and JavaScript.

## Features
- Two-player gameplay
- Winner detection
- Draw detection
- Reset Game

## Using Lang
- HTML
- CSS
- JavaScript

## How to Run
1. Download the project.
2. Open `http://127.0.0.1:5500/index.html` in your browser.
3. Start playing.
